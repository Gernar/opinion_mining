 Президент США Дональд Трамп принял решение о выходе его страны из Парижского соглашения по климату, целью которого является снижение темпов глобального потепления. Об этом сообщил в среду информационный портал Axios, который ссылается на источники, осведомленные о решении Трампа.   Механизм выхода США из соглашения разрабатывается небольшой командой, в которую входит глава Агентства по охране окружающей среды США Скотт Прюитт. Речь идет о том, чтобы инициировать полный, официальный выход из Парижского соглашения, которое может занять до трех лет, либо выход из Рамочной конвенции ООН об изменении климата, который может быть осуществлен быстрее, но, по сообщению издания, будет "более экстремальным".   Парижское соглашение заключено для выполнения Рамочной конвенции ООН об изменении климата, регулирующей меры по снижению углекислого газа в атмосфере с 2020 года. Документ, принятый в 2015 году в Париже, был подготовлен взамен Киотскому протоколу. Соглашение по климату было подписано 192 странами, 113 из которых, в том числе США, ратифицировали его.
